# modelbpp: Model BIC Posterior Probability

Fits the neighboring models of a fitted structural equation model and
assesses the model uncertainty of the fitted model based on BIC
posterior probabilities (BPP), using the method presented in Wu, Cheung,
and Leung (2020)
[doi:10.1080/00273171.2019.1574546](https://doi.org/10.1080/00273171.2019.1574546)
. See Pesigan, Cheung, Wu, Chang, and Leung (2026)
[doi:10.3758/s13428-025-02921-x](https://doi.org/10.3758/s13428-025-02921-x)
for an introduction to the package.
